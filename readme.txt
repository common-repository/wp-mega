== Description ==
WP Mega is a light but powerful plugin that can replace many plugins and make your site securer, faster, and smoother. Core features: Post Views Counter, Insert Header & Footer, Dashboard Access Control, Show or Hide Admin Bar, and so on.

Features: 
* Post / Category Views Counter
* Login/Logout Redirection
* Facebook Commenting System
* Insert Header &amp; Footer
* Dashboard Access Control
* Show / Hide Admin Bar
* Remove Version from Static Resources (CSS/JS)
* &amp; More Coming Soon

== Installation ==
1. Upload the 'wp-mega' folder to the '/wp-content/plugins' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
